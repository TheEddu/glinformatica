# GL Informática - Website

## Sobre o Projeto

Site institucional para **GL Informática**, uma empresa especializada em soluções de tecnologia da informação. O projeto apresenta os serviços oferecidos pela empresa, incluindo montagem de computadores, manutenção, infraestrutura de redes e configuração MikroTik.

O site foi desenvolvido com **HTML5, CSS3 e JavaScript** puro, apresentando uma interface responsiva e moderna com animações interativas.
