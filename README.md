# 🚀 GL Informática - Site Profissional

Um site moderno e responsivo para apresentação de serviços de tecnologia, desenvolvido com HTML, CSS e JavaScript puro.

## 📋 Características

- ✅ Design moderno e responsivo
- ✅ Navegação fluida com scroll suave
- ✅ Menu mobile com hamburger
- ✅ Seções bem organizadas (Home, Sobre, Serviços, Portfólio, Contato)
- ✅ Animações ao fazer scroll
- ✅ Formulário de contato
- ✅ Totalmente otimizado para SEO
- ✅ Compatível com todos os navegadores modernos

## 🛠️ Personalização

### Mudar Cores
Edite as variáveis CSS em `assets/css/style.css`:
```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #3498db;
    --accent-color: #e74c3c;
    /* ... */
}
```

### Adicionar Imagens
Substitua as URLs do `placeholder.com` pelas suas imagens:
```html
<img src="caminho/para/sua/imagem.jpg" alt="Descrição">
```

### Personalizar Conteúdo
Edite o arquivo `index.html` diretamente com suas informações:
- Nome da empresa
- Descrição dos serviços
- Projetos do portfólio
- Informações de contato

## 📧 Formulário de Contato

O formulário está configurado para:
- ✅ Validar campos obrigatórios
- ✅ Exibir mensagem de confirmação
- ✅ Limpar os dados após envio

Para integrá-lo com um backend, modifique o event listener em `assets/js/script.js`.
